系统简介

User Authentication

用户注册

通过手机号和短信验证码注册新用户，注册成功后自动登录

请求示例:
const url = new URL(
    "http://103.214.173.247:3019/api/auth/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "phone_area_code": "86",
    "phone": "13800138000",
    "sms_code": "123456",
    "password": "password123"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$url = 'http://103.214.173.247:3019/api/auth/register';
$response = $client->post(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'phone_area_code' => '86',
            'phone' => '13800138000',
            'sms_code' => '123456',
            'password' => 'password123',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
